本目录用于存放或标记历史阶段文档。

历史文档保留项目演进脉络，但不代表当前最新算法能力。当前主线请以以下文件为准：

- `README.md`
- `docs/current_status.md`
- `docs/current_algorithm_boundary.md`
- `code/current_3d_algorithm/`
- `outputs/latest_stable/summary.md`

如果旧文档仍保留在 `docs/` 根目录中，应把它理解为阶段记录或专题说明，而不是当前稳定算法主线。
