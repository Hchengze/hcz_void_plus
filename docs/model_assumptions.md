# Model Assumptions

## 研究对象

本项目研究城市道路浅层异常体，包括地下空洞、脱空、松散区、管沟、管线周边扰动、弱夹层和破碎带。

## 坐标和几何

- `x`：沿道路和光纤方向。
- `y`：横穿道路方向。
- `z`：深度方向，向下为正。
- 光纤近似位于 `y = 0`。
- 震源线默认位于 `y = road_width_m`。
- 异常体位于道路下方 `(x0, y0, h)`。

## 当前模型

当前 Stage 1 使用 `kinematic approximation`，只计算直达波和等效散射/绕射波的运动学走时，并叠加 Ricker 子波。

速度模型为 `uniform effective Rayleigh velocity`。这不是分层速度模型，也不包含频散、各向异性或局部速度扰动。

## 限制

当前不是完整三维弹性波全波场模拟，不能作为工程确诊结论。多个散射点表示异常体形状是运动学等效散射近似，不是真实边界散射模拟。
