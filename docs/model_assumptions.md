# Model Assumptions

## 研究对象

城市道路浅层异常体：地下空洞、脱空、松散区、管沟、管线周边扰动、弱夹层和破碎带。

## 坐标

- `x`：沿道路和光纤方向。
- `y`：横穿道路方向。
- `z / h`：深度方向，向下为正。

## Stage 2 模型

- 正演：运动学直达波 + 运动学等效散射/绕射波。
- 子波：Ricker 子波。
- 速度：`uniform effective Rayleigh velocity`。
- 伪波场：基于走时叠加的 `kinematic pseudo-wavefield snapshot`。
- 扫描：基于理论散射走时的局部能量聚焦。

## 限制

当前不是完整三维弹性波全波场模拟。伪波场和 GIF 只是传播示意，不是真实弹性波方程数值模拟。多个散射点表示异常体形状是运动学等效散射近似，不是真实边界散射模拟。
