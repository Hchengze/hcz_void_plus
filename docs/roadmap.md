# Roadmap

## Stage 1

本地科研算法工程骨架与最小 DAS-like 运动学正演闭环。

## 后续阶段

- 多炮扫描定位。
- 绕射/散射识别。
- score volume 和 best location 输出。
- 置信度评价：peak sharpness、score contrast、multi-shot consistency。
- 鲁棒性分析：噪声、坏道、速度误差、弱耦合段。
- DAS gauge length 响应增强。
- 分层速度模型。
- 局部全波场验证。

## 原则

项目是科研级算法原型，不是工业级工程软件。任何阶段都必须诚实说明近似条件，当前结果不能作为工程确诊结论。
